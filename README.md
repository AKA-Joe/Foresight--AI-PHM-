# 智瞳·工业设备健康数字孪生平台

**Aura-PHM — Predictive Health Management for Industrial Equipment**

---

## 项目背景

制造业设备管理长期受困于三重结构性矛盾：传感器数据分散于 PLC、SCADA、MES 等异构系统形成信息孤岛，传统阈值预警依赖人工经验而无法捕捉早期退化信号，算法决策缺乏可解释性导致一线运维人员"不敢信、不敢用"。叠加 5G + 边缘计算在工业场景的加速渗透，从设备侧到分析平台的数据链路安全治理已成为落地部署的前置条件。

本项目以"AI 驱动的设备全生命周期健康管理"为顶层目标，构建集**智能退化预测、数字孪生态势感知、工业网络安全巡检**于一体的技术方案，为非标设备预测性维护场景提供从算法验证到产品化的完整路径。

> **阶段说明：** 平台当前聚焦于核心算法验证与交互方案设计。鉴于工业实时数据涉及客户保密协议及数据治理合规周期，验证环境采用高保真仿真数据驱动，所有计算结果严格遵循算法逻辑链路输出。核心能力框架已完整实现，可根据具体业务场景的数据形态灵活适配与部署。

## 核心问题

1. **如何在故障发生前精准预警？** — 传统阈值规则只能在故障已显性化时触发告警，错失最佳维护窗口期。
2. **如何让 AI 判断可解释、可审计？** — 运维团队需要理解"为什么报警"而非被动接受黑盒数值，否则预测结果形同虚设。
3. **如何保障 OT 数据链路全程可信？** — 5G 边缘节点与工业网关的安全盲区可能导致数据篡改或丢失，从源头动摇分析可靠性。

---

## AI 能力

平台内置 **TADPE — 时序感知自适应退化预测引擎**（Temporal-Attentive Degradation Prediction Engine），区别于传统 LSTM/ARIMA 方案，在以下四个维度实现方法论创新：

| 创新点 | 能力描述 | 相比传统方案的优势 |
|--------|----------|-------------------|
| **多尺度时序注意力** | 自适应学习秒级振动到周级趋势的跨粒度退化模式 | 无需人工设定滑动窗口，自动聚焦关键时间尺度 |
| **物理约束正则化** | 将设备退化物理规律作为先验约束嵌入训练 | 预测曲线符合工程直觉，避免纯数据驱动的"反物理"输出 |
| **保形预测不确定性量化** | 输出带统计保证的置信区间而非单点估计 | 运维人员可量化评估"模型有多确定"，支撑风险分级决策 |
| **零样本跨设备迁移** | 架构支持跨设备类型的迁移推理 | 新设备投运即可获得基线预测，大幅降低冷启动数据门槛 |

## 项目价值

### 商业价值

- **降低非计划停机损失**：从被动抢修转为主动预防，预期减少意外停产 30%–50%，直接降低产线损失
- **优化维护资源投入**：避免过度维护带来的备件浪费与无效工时，维护成本预期下降 20%+
- **延长设备生命周期**：精准掌握退化节奏，在最佳时机介入维护，有效延长关键设备可用年限
- **赋能服务化转型**：平台能力可封装为"设备健康管理即服务"（PHMaaS）输出给下游客户，开辟新的营收模式

### 创新价值

- **算法可解释性突破**：五阶段推理流水线全程透明化，每步决策均可追溯审计，解决行业"AI 信任鸿沟"难题
- **物理-数据双驱动融合**：首创将物理退化先验与深度学习联合优化，兼顾数据拟合能力与物理一致性
- **安全-维护一体化架构**：将网络安全巡检纳入预测性维护闭环，形成"设备—算法—网络"三位一体防线，填补行业体系盲区
- **置信决策范式**：从"告警/不告警"的二值判断升级为"概率 + 区间 + 风险等级"的连续决策输出，对齐工业 4.0 精细化管理需求

### 技术价值

- 算法全链路可解释：从特征提取到最终决策，每一步可追溯
- 置信区间量化输出：为运维决策提供风险分级依据，取代"硬阈值一刀切"
- 数字孪生实时映射：设备状态从"表格数字"升级为"三维态势感知"
- 安全态势前置：OT 网络巡检左移，在数据被污染前发现链路风险

## 各模块功能说明

### TADPE 时序感知退化预测引擎

五阶段可视化算法流水线——多尺度特征提取、分层注意力编码、物理约束正则化、保形概率预测、自适应阈值决策——将"黑盒 AI"转化为"可审计的推理过程"。每阶段实时输出剩余寿命、故障概率、退化速率、置信度等关键指标，使运维团队不仅知道"设备会坏"，还能理解"为什么会坏、什么时候坏、有多大把握"。

### 5G+AI 数字孪生态势感知驾驶舱

三维设备拓扑与实时运行数据的深度融合——从全局网络拓扑到单台设备多维健康散点，管理者可在不同决策粒度间自由穿梭。流量监控、资产健康雷达、告警事件流多维联动，将分散在各系统中的设备状态聚合为"一张图看全局"的统一态势视图。

### 5G 工业网络态势感知

预测性维护的可靠性上限由底层网络决定。本模块对 5G 边缘节点、工业网关、数据传输通道执行周期性安全巡检（端口暴露面扫描、路由完整性验证、TLS/安全头合规评级），确保传感器数据"采得到、传得稳、不被篡"，补全从设备健康到网络健康的最后一块拼图。
