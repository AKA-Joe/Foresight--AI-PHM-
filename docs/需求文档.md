## 一、产品概述

### 1.1 产品定位

智瞳 Aura-PHM 是面向离散制造业非标设备（以气缸执行机构为典型对象）的预测性维护一站式平台。通过将 5G 边缘计算、时序预测算法（TADPE）、大语言模型（Claude LLM）和 3D 数字孪生技术有机融合，帮助制造企业实现从「被动维修」到「主动预测」的运维模式升级。

### 1.2 产品形态

| 形态 | 说明 |
|------|------|
| 主平台 | Windows 桌面应用（Electron），支持安装版 + 便携版 |
| 浏览器插件 | Chrome/Edge 扩展（MV3），信息采集与推送 |
| 安全工具集 | Python CLI 工具，随平台打包分发 |

### 1.3 产品边界

| 范围内 | 范围外 |
|--------|--------|
| 设备退化趋势预测与可视化 | 设备自动控制与执行 |
| AI 辅助运维分析报告生成 | 最终维修决策（人工判断） |
| 5G 网络健康监测与安全巡检 | 5G 网络的运营商侧管理 |
| 3D 数字孪生展示 | CAD 级别设备建模 |
| 历史数据分析 | 实时 PLC 控制指令下发 |

## 五、数据模型

### 5.1 核心实体关系

```
CylinderStatus (气缸设备)
├── uid: string (主键, 格式: CSKG-LINE-STATION-FUNCTION-CYLNN)
├── name: string (中文名称)
├── line: string (产线)
├── station: string (工位)
├── device: string (设备)
├── baselineMs: number (基线动作时间)
├── fixedThreshold: number (固定阈值 = baseline × 1.4)
├── dynamicUpperMs: number (动态上阈值)
├── latestMs: number (最近动作时间)
├── healthScore: number (0-100)
├── faultProbability: number (0-100)
└── alertLevel: enum (normal | info | warning | critical)

Alert (告警记录)
├── id: string
├── cylinderUid: string (FK → CylinderStatus)
├── level: enum (critical | warning | info)
├── title: string
├── timestamp: string (ISO 8601)
└── status: enum (active | acknowledged | closed)

MaintenanceRecord (维护记录)
├── id: string
├── cylinderUid: string (FK → CylinderStatus)
├── type: enum (preventive | corrective | emergency)
├── operator: string
├── timestamp: string (ISO 8601)
└── result: string

TimeSeriesPoint (时序点)
├── cylinderUid: string (FK → CylinderStatus)
├── timestamp: string
├── valueMs: number (动作执行时间)
├── baseline: number
├── dynamicThreshold: number
└── fixedThreshold: number

DashboardKPI (仪表盘指标)
├── totalCylinders: number
├── avgHealth: number
├── activeAlerts: { critical, warning, info }
├── pendingMaintenance: number
└── dataQuality: number (百分比)
```

### 5.2 设备命名规范

```
CSKG - LA - ST01 - PUSH - CYL01
  │      │     │      │      │
  │      │     │      │      └── 气缸序号 (01-99)
  │      │     │      └── 功能代码
  │      │     │           PUSH  = 推送
  │      │     │           PULL  = 拉回
  │      │     │           LIFT  = 升降
  │      │     │           CLAMP = 夹紧
  │      │     │           ROTATE = 旋转
  │      │     │           LOCK  = 锁定
  │      │     └── 工位号 (ST01-ST99)
  │      └── 产线代码 (LA=A线, LB=B线, LC=C线, LD=D线)
  └── 工厂代码 (CSKG)
```

### 5.3 运行模式分类

| 模式 | 特征 | 对应设备(示例) | 健康评分范围 |
|------|------|----------------|--------------|
| normal | 稳定在基线附近 | CYL-01 | 85-100 |
| drift | 缓慢上升趋势 | CYL-03 | 65-80 |
| critical | 显著超阈值 | CYL-07 | 20-40 |
| volatile | 大幅波动 | CYL-09 | 55-75 |
| recovered | 维护后恢复 | CYL-12 | 70-85 |
| dirty | 数据含噪声 | CYL-15 | 80-95 |
