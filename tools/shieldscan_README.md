## ✨ 功能特性

| 模块 | 功能 | 可视化效果 |
|------|------|-----------|
| 🔒 **安全头检测** | 检测 12 项 HTTP 安全头（HSTS/CSP/XFO/XSS 等） | 彩表（✅优秀/⚠️基本/❌缺失） |
| 📜 **TLS 审查** | 证书主题、签发者、有效期、SANs、协议版本 | 有效期进度条 + 安全色标 |
| 🍪 **Cookie 分析** | HttpOnly/Secure/SameSite 标志检查 | ✅/❌ 标志表 |
| 🏆 **综合评分** | 加权评分 + A+~F 六级评级 | 色条 + 各子项得分 |
| 📋 **改进建议** | 每项未达标配置给出修复方案 | 严重程度排序（🔴高危/🟡中危） |
| 📊 **扫描总览** | 汇总卡片 | 安全头达标数/TLS/Cookie/评分/用时 |
| 📄 **HTML 报告** | 暗色主题完整报告 | 评分大徽章 + 统计卡片 + 表格 |

## 🚀 用法

### 命令行模式

```bash
# 快速扫描
python shieldscan.py https://example.com

# 自动补全 HTTPS
python shieldscan.py example.com

# 导出 HTML 报告
python shieldscan.py example.com --html report.html
python shieldscan.py https://example.com -o scan.html
```

### 交互式模式

```bash
python shieldscan.py
```

进入菜单引导：
1. 输入目标网站 URL
2. 自动执行全扫描：安全头 → TLS 证书 → Cookie 分析 → 评分
3. 选择是否导出 HTML 报告

## 🧠 评分体系

| 评级 | 分数 | 含义 | 标签 |
|:----:|:----:|------|------|
| **A+** | 95-100 | 卓越 | 🏆 无懈可击 |
| **A** | 85-94 | 优秀 | 🛡️ 轻微改进空间 |
| **B** | 75-84 | 良好 | 👍 建议增强 |
| **C** | 60-74 | 及格 | ⚠️ 存在明显风险 |
| **D** | 40-59 | 薄弱 | 🔴 多处需改进 |
| **F** | 0-39 | 危险 | 💀 极易受攻击 |

评分 × 权重体系：
- **HSTS/CSP/XFO** — 权重高 (15-20)，核心安全机制
- **MIME/XSS/Referrer** — 权重中 (5-10)，纵深防御
- **TLS 证书** — 额外加分项 (最多 +10)

## 📄 HTML 报告

导出为深色主题 HTML，包含：
- **评级大徽章** — A+~F 带颜色边框 + 分数
- **统计卡片组** — 检测项 / 通过 / 未通过 / Cookie / 用时
- **TLS 信息卡** — 签发者 / 有效期 / 协议 / 加密套件
- **安全头表格** — 每个头的状态 + 当前值
- **Cookie 表格** — HttpOnly / Secure / SameSite 三标志
- **改进建议** — 每项未达标配置的修复方案
